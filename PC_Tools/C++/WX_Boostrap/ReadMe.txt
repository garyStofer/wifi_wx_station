Extract the files in this zip file to a location on your harddrive and make a shortcut to WX_Bootstrat.exe 
that you can put on the desktop for eaxmple.

The file TCPIP_Discovery.jar is a java program that displays the Netbioas name and the IP address of a WX_station
board after it's connected to the network. The Java Run time must be installed on the computer to run it.
