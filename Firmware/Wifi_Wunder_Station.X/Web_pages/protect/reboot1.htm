Reboot

Your board is located at: http://~rebootaddr~

To restore factory default settings proceed as follows:

  • Disconnect power to the device.
  • Press and hold button S1.
  • Apply power while holding Button S1.
  • Wait until 3 LEDs flash rapidly after about 5 seconds.
  • Release the button.

The device will re-start with the following connection defaults:

  1. WIFI SSID : linksys
  2. WIFI encryption : None
  3. Netbios name : WXBOARD
  4. DHCP : Enabled
  5. Login Username : admin
  6. Login Password : admin

During the boot process the device displays the current connection parameters via its USB COM port. Use a terminal emulator program configured to 19.2kbd, 8N1 to connect
