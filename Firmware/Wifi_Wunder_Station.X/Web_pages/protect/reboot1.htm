Your board is located at: http://~rebootaddr~

To restore factory default settings proceed as follows:

  • Reset the device while pressing switch S1.
  • You can reset via the Reboot Now button, by cycling power, or by momentary connecting pin 1 & 3 on JP1
  • Wait until the 3 LEDs flash rapidly, after about 4-5 seconds.
  • Release switch S1.

The device will re-start with the following connection defaults:

  1. WIFI SSID : linksys
  2. WIFI encryption : None
  3. Netbios name : WXBOARD
  4. DHCP : Enabled
  5. Login Username : admin
  6. Login Password : admin

To configure WiFi and TCP/IP settings via the USB link proceed as follows:

  • Connect the device to a PC via the USB link.
  • Using a terminal emulator configured to 38.4kbd,8N1, connect to the appropriate COM port.
  • While pressing switch S1, Reset the device by momentarily connecting JP1 pin 1 & 3.
  • Release switch S1 after the initial flash of the 3 LEDs.
  • A menu is displayed that allows you to configure the essential WiFi and TCP/IP settings.
