To restore factory settings proceed as follows:
  • Press and hold switch S1
  • Press the RESET switch momentarily or cycle power.
  • Wait 5 seconds, until the 3 LEDs flash rapidly.
  • Release switch S1.

The device will re-start with the following defaults:

  1. WIFI SSID : linksys
  2. WIFI encryption : None
  3. Netbios name : WXBOARD
  4. DHCP : Enabled
  5. Login Username : admin
  6. Login Password : admin
To configure WiFi & TCP/IP settings via USB proceed as follows:
  • Connect the device to a PC via the USB link, see device driver footnote.
  • Verify that the USB COM port has been added to the system.
  • Connect to the COM port with a terminal emulator set to 38.4Kbd,8N,1,NoHS.
  • Press and hold switch S1
  • Press the RESET switch momentarily or use the "Reboot Now" button.
  • Release switch S1 after the initial flash of the 3 LEDs.
  • A menu is displayed that allows you to configure the essential WiFi & TCP/IP settings.

† In order for your PC to connect to the USB port you need to install:   USB Device Driver
